# Gaia Jovik

## Come sviluppare

### Requisiti

* Python 3 (es. `python3`)
* PIP 3 (es. `pip3`)
* MySQL 5.5+ (o MariaDB 10)

### Dipendenze

```bash
sudo apt-get install python3
sudo apt-get install python3-pip
```
### Installazione

Dopo aver clonato il progetto,

```bash
cd jorvik/
sudo pip3 install -r requirements.txt
```
